## Repurpose playbook
### Pillar article → LinkedIn
- 3 posts: opening hook, key stat, contrarian take
- Each post 150-300 words, native format (no link in main text)

### Pillar article → X thread
- 1 thread: 8-12 tweets
- Tweet 1: hook + promise
- Tweets 2-11: one insight per tweet
- Tweet 12: CTA + link to full article

### Pillar article → Instagram carousel
- 8-10 slides
- Slide 1: hook (bold text on background)
- Slides 2-9: one key point per slide
- Slide 10: CTA

### Pillar article → TikTok
- 60-90 sec script
- Hook in first 3 seconds
- 1 key insight, no fluff
- CTA: "Full breakdown in bio"

### Pillar article → YouTube Short
- 60 sec vertical
- Same script as TikTok, slight re-edit
- CTA: subscribe + link in description
