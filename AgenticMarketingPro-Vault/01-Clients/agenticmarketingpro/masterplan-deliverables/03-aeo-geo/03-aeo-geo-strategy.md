## 2. LLM Citation Strategy (Platform-by-Platform)

### 2.1 ChatGPT Citation Playbook

**How ChatGPT cites:**
- Bing search results (browsing mode)
- Plugin/web search retrievals
- Training data (pre-cutoff)
- Custom GPT configurations

**Tactics:**
1. **Bing-first SEO** — ChatGPT's browsing uses Bing index. Achieve top 3 Bing rankings for "agentic marketing" cluster keywords.
2. **OpenAI Plugin/GPT Store** — Build a custom GPT ("Agentic Marketing Strategist") referencing AMP content; cite your own domain.
3. **High-authority mentions** — Get quoted/featured in outlets ChatGPT was trained on (TechCrunch, Forbes, HubSpot, MarketingProfs).
4. **Structured Q&A content** — Mirror common ChatGPT prompts verbatim as H2/H3 headings on your site.

### 2.2 Perplexity Citation Playbook

**How Perplexity cites:**
- Real-time web crawl (primary)
- Heavy preference for recency and authority
- Strong recency bias (publishing cadence matters)

**Tactics:**
1. **Daily publishable cadence** — Aim for 3-5 fresh indexed pages/week.
2. **Citation-worthy formatting** — Author bio, publish date, sources cited inline.
3. **Listicle + stat headline patterns** — "10 Agentic Marketing Statistics [2025]" formats cite well.
4. **Direct answer paragraphs** — Lead each section with a 40-60 word direct answer to the implicit question.

### 2.3 Gemini Citation Playbook

**How Gemini cites:**
- Google Knowledge Graph (highest trust)
- Google's index (freshness matters)
- YouTube transcripts
- Google Scholar / Books

**Tactics:**
1. **YouTube content parity** — Every pillar page gets a matching YouTube video with full transcript on the page.
2. **Google Business Profile / Knowledge Panel** — Establish founder and company panels.
3. **Google Scholar papers** — Co-author or commission whitepapers that appear in Scholar index.
4. **AMP (Accelerated Mobile Pages) / Core Web Vitals** — Gemini's underlying index uses Google ranking signals.

### 2.4 Claude Citation Playbook

**How Claude cites:**
- Anthropic's curated training (preference for quality publications)
- Web fetch when enabled
- Indirect citations via tools

**Tactics:**
1. **Long-form depth** — Claude favors substantive analysis over thin listicles. 2,500+ word pillar content.
2. **Nuanced, balanced framing** — Claude's training biases toward considered viewpoints, not pure marketing copy.
3. **PDF whitepapers** — Publish authoritative PDFs that get cited as evidence in Claude's outputs.

### 2.5 Microsoft Copilot Citation Playbook

**How Copilot cites:**
- Bing index (primary)
- LinkedIn content (heavy integration)
- Microsoft Start network

**Tactics:**
1. **LinkedIn-first distribution** — Native articles, founder posts, company updates. Copilot surfaces LinkedIn content frequently.
2. **Bing Webmaster Tools** — Submit all pillar content. Use IndexNow API for instant indexing.
3. **MSN/Microsoft Start submission** — Apply for publisher partnership; content appears in Copilot responses.

### 4.1 The "Perplexity Citation Triggers"

Perplexity preferentially cites content with these attributes:
1. **Recency** — Content published within 90 days ranks 3x more often
2. **Author authority** — Bylined articles with author bios and LinkedIn links
3. **Inline statistics** — Numeric claims with source attribution
4. **Direct answer paragraphs** — 40-60 word passages immediately following H2/H3
5. **Structured data** — Schema.org FAQPage, Article, Organization
6. **External citations** — Articles that cite other authoritative sources get cited more
7. **Listicle formats** — "X tools," "Y ways," "Z examples"
8. **Comparison tables** — Extracted directly into responses
9. **TL;DR/summary boxes** — Perplexity extracts these as answer snippets
10. **Authoritative domain (.edu, .org, established .com)** — Not optional

## 5. LLM Prompt Testing Protocol (50 Variations/Week)

### 5.1 Testing Framework Structure

**Test categories (10 prompts per category, run weekly):**

| Category | Prompt Pattern | Frequency |
|----------|---------------|-----------|
| Definitional | "What is [agentic marketing term]?" | 10 prompts |
| Comparative | "[Term A] vs [Term B]" | 10 prompts |
| Vendor/Tool | "Best [category] tools/platforms 2025" | 10 prompts |
| Statistical | "[Category] statistics/data/benchmarks 2025" | 10 prompts |
| How-To | "How to [achieve outcome with agentic marketing]" | 10 prompts |
| **Total weekly** | | **50 prompts** |

### 5.2 Test Execution Protocol

**Step 1: Prompt Library (build once, refresh quarterly)**
Maintain a master prompt library of 500+ verified prompts covering:
- Category-defining queries
- Buyer-journey queries (awareness → consideration → decision)
- Practitioner queries
- Comparative queries
- Objection-handling queries

**Step 2: Weekly testing schedule**
- Monday: Definitional batch (10 prompts × 5 LLMs = 50 responses)
- Tuesday: Comparative batch
- Wednesday: Vendor/Tool batch
- Thursday: Statistical batch
- Friday: How-To batch

**Step 3: Response capture**
For every prompt, capture:
- Full LLM response (verbatim)
- All cited sources (URLs, domains)
- Position of AMP citation (1st, 2nd, 3rd, etc.)
- Citation context (is AMP cited favorably, neutrally, or unfavorably?)
- Competitor citations
- Competitor mentions without citation
- Knowledge graph entities surfaced

**Step 4: Scoring rubric**

```
Citation Scoring (per response):
- AMP cited as #1 source: 10 points
- AMP cited in top 3: 7 points
- AMP cited anywhere: 5 points
- AMP mentioned without citation: 3 points
- Competitor cited as #1: 0 points
- No AMP reference: 0 points

Weekly Citation Score = Sum of all 250 responses
```

**Step 5: Action triggers**
- If citation rate drops >10% WoW → content audit trigger
- If a competitor overtakes AMP on a specific prompt → competitive response content within 7 days
- If a new prompt emerges as high-volume → add to pillar content backlog

### 5.3 Required Tooling

**Recommended stack:**
- **Prompt testing:** Custom Python script using OpenAI API, Anthropic API, Google AI Studio API, Perplexity API
- **Citation extraction:** Diffbot or similar entity extraction
- **Tracking dashboard:** Airtable or Notion database with weekly rollup
- **Alerting:** Slack/email alerts on score drops or new competitor citations
