## 🎯 Purpose & Scope

This SOP establishes a systematic, repeatable process for identifying, prioritizing, and executing content refreshes across AgenticMarketingPro's owned media properties. The goal is to **prevent organic content decay**, **maximize ROI on existing content assets**, and **maintain topical authority** in the agentic AI marketing space.

**Scope:** All published content including blog posts, pillar pages, landing pages, case studies, whitepapers, and gated assets.

### 📅 Audit Cadence

| Quarter | Audit Focus | Lead Owner |
|---------|-------------|------------|
| Q1 | Full content inventory + decay scan | Content Strategist |
| Q2 | Performance refresh + competitor gap | SEO Specialist |

### 🔍 Phase 1: Pre-Audit Preparation (Week 1)

**Tasks:**

### 🔍 Phase 2: Audit Execution (Week 2)

**Tasks:**
- [ ] Score every published page across 5 dimensions (see Section 2)
- [ ] Identify decay signals (Section 3)

- [ ] Flag outdated statistics, broken links, deprecated product mentions
- [ ] Identify content cannibalization issues
- [ ] Document content gaps vs. current keyword landscape

**Tasks:**
- [ ] Categorize content into 4 quadrants (Refresh, Maintain, Consolidate, Retire)
- [ ] Build refresh priority matrix (Section 4)
- [ ] Estimate effort hours per refresh candidate
- [ ] Create quarterly refresh roadmap with 8-12 prioritized pieces
- [ ] Present findings to Content Director for approval

## SECTION 2: CONTENT DECAY DETECTION SIGNALS & METRICS

### 🚨 Primary Decay Signals

| Signal | Threshold | Detection Method |
|--------|-----------|------------------|
| Organic traffic decline | >20% drop over 6 months | GA4 / Search Console |
| Keyword ranking loss | Lost top 10 position for target KW | Ahrefs / SEMrush |
| Click-through rate decline | <2% CTR on page 1 rankings | Search Console |
| Backlink decay | >10% referring domains lost | Ahrefs |
| Content age | >18 months without update | CMS metadata |
| Impressions drop | >30% YoY impressions decline | Search Console |

### 📊 Content Health Scoring (5 Dimensions)

Score each piece **1–5** (5 = excellent, 1 = critical):

| Dimension | Scoring Criteria |
|-----------|------------------|
| **Traffic Performance** | Current vs. peak traffic, trend direction |
| **Ranking Health** | Position 1-3, 4-10, 11-20, 21+ for target KW |
| **Conversion Performance** | CTA clicks, form fills, pipeline contribution |
| **Topical Relevance** | Alignment with current product/service offerings |
| **Technical Quality** | Page speed, mobile UX, schema, internal links |

**Composite Score:** Average of 5 dimensions
- **4.0–5.0** → Maintain
- **3.0–3.9** → Monitor
- **2.0–2.9** → Refresh Candidate
- **<2.0** → Major Rewrite or Retire

## SECTION 3: REFRESH PRIORITY MATRIX

### 🎯 Impact vs. Effort Quadrant

| | **Low Effort** | **High Effort** |
|---|---|---|
| **High Impact** | 🟢 **QUICK WINS** (Tier 1) | 🟡 **STRATEGIC PROJECTS** (Tier 2) |
| **Low Impact** | ⚪ **FILL-INS** (Tier 3) | 🔴 **RE-EVALUATE** (Tier 4) |

### Tier Definitions

| Tier | Description | Action | SLA |
|------|-------------|--------|-----|
| **Tier 1: Quick Wins** | High-traffic pages with minor decay | Refresh in 2 weeks | 2 weeks |
| **Tier 2: Strategic** | Pillar pages or high-converting assets | Full refresh in 4 weeks | 4 weeks |
| **Tier 3: Fill-Ins** | Minor updates to supporting content | Batch updates monthly | 4 weeks |
| **Tier 4: Re-evaluate** | Consider consolidation or retirement | Decision in next audit | N/A |

### Prioritization Formula

```
Priority Score = (Traffic Value × 0.3) + (Conversion Value × 0.3) 
               + (Strategic Fit × 0.2) + (Ease of Refresh × 0.2)
```

## SECTION 4: STEP-BY-STEP REFRESH WORKFLOW

### Stage 1: Research & Planning (Days 1–3)

- [ ] Identify refresh candidate from priority matrix
- [ ] Conduct keyword gap analysis vs. current top-ranking competitors
- [ ] Review Search Console for query opportunities
- [ ] Identify new subtopics or sections to add
- [ ] Check for new statistics, examples, or case studies to integrate
- [ ] Outline refresh scope in project management tool

### Stage 2: Content Refresh (Days 4–8)

- [ ] Update title tag and meta description (incorporate new KW opportunities)
- [ ] Refresh H1 and restructure headings if needed
- [ ] Update or expand body content (target: 20–40% new/updated material)
- [ ] Add new sections addressing identified content gaps
- [ ] Update all statistics, dates, and product references
- [ ] Add 2–3 new internal links to relevant current content
- [ ] Add 1–2 authoritative external links (recent sources)
- [ ] Refresh images, screenshots, and visual assets
- [ ] Add/update schema markup (Article, FAQ, HowTo as applicable)

### Stage 3: Optimization (Days 9–10)

- [ ] Run content through SEO optimization tool (Surfer, Clearscope, Frase)
- [ ] Verify target keyword density and semantic coverage
- [ ] Check readability score (target: Grade 8-10 for B2B)
- [ ] Optimize for featured snippets (answer boxes, lists, tables)
- [ ] Add or refresh CTA (current offer alignment)
- [ ] Verify all links work (internal, external, CTAs)

### Stage 4: QA & Publication (Days 11–12)

- [ ] Editorial review (grammar, tone, brand voice)
- [ ] SEO technical review (canonical, schema, redirects if URL changed)
- [ ] Legal/compliance review for regulated content
- [ ] Update publish date in CMS metadata
- [ ] Add "Last Updated: [Date]" visible on page
- [ ] Submit updated URL to Google Search Console (if significant changes)
- [ ] Promote via owned channels (email, social, Slack)

### Stage 5: Post-Refresh Monitoring (Days 13–90)

- [ ] Monitor rankings for target keywords weekly
- [ ] Track traffic recovery vs. pre-refresh baseline (Day 1)
- [ ] Document wins in refresh log
- [ ] Set 90-day review checkpoint
- [ ] Iterate if performance doesn't improve

## SECTION 5: CONTENT UPDATE CHECKLIST

### 📝 Universal Refresh Checklist

**Metadata**
- [ ] Title tag updated (55–60 characters)
- [ ] Meta description updated (150–160 characters)
- [ ] URL slug reviewed (update only if necessary; use 301 redirect)
- [ ] Open Graph image refreshed
- [ ] Canonical tag verified

**Content Quality**
- [ ] All statistics updated to current year data
- [ ] All product mentions reflect current offerings
- [ ] Screenshots reflect current UI/product
- [ ] Author bio updated if applicable
- [ ] Examples are current and relevant
- [ ] No references to outdated events or trends

**SEO**
- [ ] Target keyword in H1, first 100 words, and 1–2 subheadings
- [ ] Internal links added to 2–3 newer relevant pages
- [ ] 1–2 outbound links to authoritative 2024–2025 sources
- [ ] Image alt text updated and descriptive
- [ ] Schema markup validated
- [ ] FAQ section added/refreshed (for featured snippet targeting)

**UX & Technical**
- [ ] Page load time <3 seconds
- [ ] Mobile experience verified
- [ ] CTA above fold and contextual
- [ ] No broken links (internal or external)
- [ ] Table of contents added if content >1,500 words

### 📊 Refresh Impact Scorecard

```
ROI Calculation:
Gain Value = (New 90-day traffic × Conversion rate × Lead value) - Old 90-day value
Refresh Cost = (Hours × Hourly rate) + Tools + Promotion
Net ROI = Gain Value - Refresh Cost
```

**Target:** Minimum 3:1 ROI within 6 months of refresh.
