# Link Prospects

> Running list of link opportunities, scored by DR + relevance + likelihood. Owned by Off-Page Strategist.

## Prospect schema
| # | Domain | DR | Relevance (1-5) | Likelihood (1-5) | Score | Type | Contact | Status | Last contacted |
|---|---|---|---|---|---|---|---|---|---|
| | | | | | | | | | |

## Score formula
`Score = DR × Relevance × Likelihood` (max = 25 × 5 × 5 = 625)

## Prospect types
- **Resource page:** site has a "resources" or "tools" page where our content fits
- **Guest post:** site accepts guest contributions
- **Broken link:** site links to a dead URL we can replace
- **Skyscraper:** site links to inferior content on same topic
- **Digital PR:** journalist or blogger who covers our industry
- **HARO/Qwoted:** journalist query we can respond to
- **Podcast:** podcast host who'd have our client on
- ** Partnership:** complementary brand for co-marketing

## Forbidden domains (never prospect)
- PBNs (private blog networks)
- Link farms (sites that exist only to sell links)
- Irrelevant foreign-language sites
- Sites with spam score >30 (Moz)
- Sites with DR <15 (low ROI)
- Competitor websites (no, really — agent should reject)

## Outreach cadence
- New prospects added weekly
- Top 10 prospects contacted per client per week
- Follow-up cadence: 3 days, 7 days, 14 days, then archive
