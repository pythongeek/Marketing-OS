# Keyword Universe

> Master keyword list across all clients. Grouped by intent. Auto-enriched by Topic Expert + Content Strategist agents.

## How to use this file
- One row per keyword
- Tags link keywords to clients (so we can query "all commercial-intent keywords for Acme")
- Topic cluster assignment drives internal linking strategy

## Intent categories
- **Informational:** "what is X", "how to X", "X vs Y" — top-of-funnel, content marketing
- **Commercial:** "best X", "X reviews", "X alternatives" — mid-funnel, comparison/buying guides
- **Transactional:** "buy X", "X pricing", "X demo" — bottom-funnel, conversion pages
- **Navigational:** brand queries — owned by homepage/landing pages
