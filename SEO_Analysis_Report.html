7GSC Configuration & Setup Instructions

⚙️ How to Set Up GSC API for Ongoing Analysis

7.1 Why You Need GSC API Access

  • Export unlimited query data (vs 1,000 row limit in UI)
  • Automated weekly reports
  • Regex filtering for pattern discovery
  • Historical data comparison

7.2 Step-by-Step Setup

Step 1: Create Google Cloud Project

  1. Go to Google Cloud Console
  2. Create new project: "GSC-Analysis"
  3. Enable "Search Console API"

Step 2: Create Credentials

  1. Go to APIs & Services → Credentials
  2. Create OAuth 2.0 Client ID
  3. Download JSON credentials file
  4. Add your email as test user

Step 3: Verify Property in GSC

  1. Go to Search Console
  2. Add property: agenticmarketingpro.com
  3. Verify via DNS or HTML upload
