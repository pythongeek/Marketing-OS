Tier 1 — Strategy & Intelligence agents
Competitor Intel Agent
Crawls rivals weekly. Tracks their content, backlinks, keyword gains, social, ad spend signals, and SERP movements. Auto-populates competitor map in Obsidian.
Revenue Channel Scout
Identifies untapped revenue channels: affiliate, white-label, retainer, productized services, consulting upsells. Scores each by effort vs. ROI potential.
Analytics Expert Agent
Owns GA4, Looker Studio, attribution. Generates weekly ROI reports, channel mix analysis, conversion funnel insights, and anomaly alerts automatically.
Topic Authority Agent
Maps full topical coverage gaps using semantic clustering. Builds pillar-cluster architecture. Flags cannibalisation, missing E-E-A-T signals, and entity gaps.
Profit Plan Agent
Monthly P&L projection agent. Models client acquisition cost, LTV, service margins, and recommends next-best actions to hit revenue targets.
Tier 2 — Content & SEO production agents
Content Strategist
Runs the editorial calendar. Assigns briefs, sets intent targets, manages the cluster hierarchy, and sequences content for max topical authority build-up.
Long-form Writer Agent
Writes 2,000–5,000 word authority articles, pillar pages, and case studies. Trained on your brand voice. Uses research tools and cites real data.
Copywriting Agent
Landing pages, CTAs, email sequences, ad copy, sales pages, cold outreach. Multi-framework specialist: PAS, AIDA, StoryBrand, direct response.
Programmatic SEO Agent
Generates 100s–1000s of unique location+service, comparison, and tool pages from structured data templates. Owns the pSEO pipeline from data → CMS publish.
AEO / GEO Specialist
Optimises every page to be cited in ChatGPT, Perplexity, Google AI Overviews, Copilot. Deploys FAQ schema, entity markup, expert quotes, statistic injection.
Writer Persona Agents (×5)
Data-journalist · Thought-leader · Conversational · Technical · Storyteller. Matched to content type and target audience for brand-consistent variety.
Tier 3 — On-page, off-page & technical SEO agents
On-Page Optimisation Agent
Audits and rewrites title tags, meta descriptions, H1–H3 structure, internal links, schema markup, image alt text, and content semantic density on every published page.
Off-Page / Link Agent
Prospecting, outreach sequences, HARO/digital PR, broken-link reclamation, competitor backlink gap analysis, and monthly link velocity reporting.
Google Search Console Agent
Daily GSC monitoring. Detects CTR drops, index coverage errors, manual actions, Core Web Vitals regressions, and generates fix-priority action lists.
Bing / Copilot SEO Agent
Manages Bing Webmaster Tools, optimises for Bing-specific ranking factors, and ensures Copilot AI visibility — an underserved, lower-competition channel.
Site Health Monitor
24/7 uptime, crawl budget, Core Web Vitals, broken links, redirect chains, hreflang errors, and log file analysis. Fires alerts and creates tickets automatically.
Technical SEO Agent
Crawls site architecture, JavaScript rendering, structured data validity, page speed, mobile usability, and log analysis. Monthly technical health report.

Ad Expert Agent

Email Marketing Agent

Lead Generation Agent
